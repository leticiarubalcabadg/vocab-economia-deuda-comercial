Vocabulario para la representación de la información de la Deuda Pública Comercial de un ayuntamiento: Descripción volver a índice

La Figura 1 muestra las clases y propiedades del vocabulario que representa la Deuda Pública Comercial de un ayuntamiento:

En este vocabulario se tienen clases para representar el [esdeucom:InformeMorosidadTrimestral](#InformeMorosidadTrimestral) y el [esdeucom:InformeMorosidadMensual](#InformeMorosidadMensual) . Las propiedades de estas dos clases se corresponden a los campos de cada uno de los informes destacando el [esdeucom:periodoMedioPagoMensual](#periodoMedioPagoMensual) o PMP mensual y el [esdeucom:periodoMedioPagoTrimestral](#periodoMedioPagoTrimestral) o PMP trimestral. Los informes de morosidad mensual y trimestral se realizan en un determinado [esdeucom:periodo](#periodo) de tal manera que en es posible representar los informes de morosidad de varios meses o de varios trimestres y así poder realizar análisis comparativos acerca de la deuda comercial de un ayuntamiento a lo largo del tiempo. Además se tiene que el PMP Mensual se calcula globalmente para el ayuntamiento y se representa mediante la propiedad [esdeucom:periodoMedioPagoMensualGlobal](#periodoMedioPagoMensualGlobal) dentro de la clase [esdeucom:InformeMorosidadMensualGlobal](#InformeMorosidadMensualGlobal). Para representar el período se ha reutilizado la ontología [time](https://www.w3.org/TR/owl-time/), en particular las clases [time:ProperInterval](#http://www.w3.org/2006/time#ProperInterval) e [time:Instant](#http://www.w3.org/2006/time#Instant) para representar el inicio y final de un período. Los informes de morosidad se realizan tanto para la entidad local en sí como para sus entidades dependientes. La entidad local y sus entidades dependientes se representan mediante la clase [org:Organization](https://www.w3.org/TR/vocab-org/#class-organization) de la ontología [Organization](https://www.w3.org/TR/vocab-org/).

3.1 Ejemplos de uso volver a índice

Con la finalidad de facilitar la comprensión del uso de algunas clases y propiedades de este vocabulario se proporcionarán a continuación algunos ejemplos. El primer ejemplo se refiere al informe de morosidad trimestral de una entidad dependiente del ayuntamiento de Madrid, en particular se ejemplifica la Empresa Municipal de Transportes de Madrid para el cuarto trimestre del 2019.

        esdeucom:periodo <http://vocab.ciudadesabiertas.es/recurso/economia/deuda-comercial/intervalo-tiempo/2019-cuarto-trimestre>;
        esdeucom:periodoMedioPagoTrimestral "55.50"^^xsd:float ;
        esdeucom:numPagosDentroPeriodo "6472"^^xsd:integer ;
        esdeucom:importePagosDentroPeriodo "51831754.13"^^xsd:float ;
        esdeucom:numPagosFueraPeriodo "1124"^^xsd:integer ;
        esdeucom:importePagosFueraPeriodo "5579630.33"^^xsd:float ;
        esdeucom:numPagosInteresesDemora "0"^^xsd:integer ;
        esdeucom:importePagosInteresesDemora "0.0^^xsd:float ;
        esdeucom:periodoMedioPagoPendiente "33.87^^xsd:float ;
        esdeucom:numFacturasPendientesDentroPeriodo "3020"^^xsd:integer ;
        esdeucom:importeFacturasPendientesDentroPeriodo "34441734.41^^xsd:float ;
